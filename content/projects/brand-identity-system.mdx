# A Journey through Middle-Earth

Immersive website experience exploring Tolkien's Middle-Earth. Created as final project for Immersive Experiences Design class.

## Challenge

Design an engaging digital experience that captures the wonder of Middle-Earth while maintaining intuitive user interaction.

## Solution

Created a narrative-driven website that guides users through iconic locations using:
- Atmospheric imagery and typography
- Scroll-triggered animations
- Interactive hotspots
- Smooth transitions between sections

## Process

Built with Framer to leverage advanced prototyping capabilities. Focused on balancing visual impact with usability across all devices.

## Key Features

- Immersive storytelling through guided chapters
- Interactive elements that enhance the narrative
- Responsive design for all screen sizes
- Optimized performance despite rich visual content

## Results

Well-received class presentation with praise for seamless storytelling integration and thoughtful animation use.

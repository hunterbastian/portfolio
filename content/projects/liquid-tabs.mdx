## The Challenge

Build two tab navigation variants — a pill indicator and an underline indicator — both sliding with an elastic spring, in 15 minutes. No animation libraries. No dependencies. Just CSS transitions and a few lines of DOM math.

## The Trick

A single absolutely-positioned `<div>` gets repositioned with `getBoundingClientRect()` on click. The `left` and `width` properties transition independently with different durations, so the indicator subtly stretches mid-travel before snapping to its final size. The easing is `cubic-bezier(.34, 1.56, .64, 1)` — an elastic overshoot that feels like a spring without needing JS animation loops.

Panel content swaps use a `position: absolute` → `relative` swap so the entering panel pushes layout while the exiting panel collapses without a flash.

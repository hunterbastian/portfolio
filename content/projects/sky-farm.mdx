## Overview

Sky Farm is a browser-based idle farming game rendered entirely with procedural pixel art. A floating island above the clouds hosts crops, chickens, cherry blossom trees, and a pond — all growing and harvesting on their own.

## Direction

<ScrollReveal>

The visual style draws from Stardew Valley and early-2000s Windows XP aesthetics — soft CRT scanlines, a Luna-themed toolbar, and a DOS-style boot sequence. Everything is drawn procedurally each frame with no sprite sheets.

</ScrollReveal>

## What It Explores

<ScrollReveal>

Auto-farm mechanics, touch-first interaction design, real-time day/night lighting, and the challenge of building a complete game loop in a single TypeScript file with PixiJS v8.

</ScrollReveal>
